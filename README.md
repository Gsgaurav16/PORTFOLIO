### Prerequisites

- Node.js (v16 or higher)

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

```bash

```

The built files will be in the `dist` directory.
